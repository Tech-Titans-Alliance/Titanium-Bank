# Titanium Bank 💳🏦

## Project Overview
Titanium Bank is a fully functional banking system built using Java and JavaFX. This system enables users to perform a wide range of banking activities such as creating an account, depositing and withdrawing money, transferring funds, viewing accounts, and checking transaction history. It also supports overdraft management, ensuring a comprehensive banking experience.

## Features 🌟
- **Register & Login:** Users can create a new account or log in using their existing credentials
- **Create Account:** Users can create a new bank account after registering.
- **Withdraw Money:** Users can withdraw funds from their account.
- **Transfer Money:** Users can transfer money between different accounts.
- **Deposit Funds:** Users can deposit money into their account.
- **View Accounts:** Users can view all their bank accounts.
- **View Transaction History:** Users can view detailed transaction history for each account.
- **Overdraft Management:** Users can manage their overdraft limits for a flexible banking experience.
- **Version Control: Git** (for code management and collaboration)

## Technologies Used 💻
- **Backend:** Java
- **Frontend:** JavaFX (FXML)
- **Database:** In-memory storage (for simplicity)

## Getting Started 🚀

### Prerequisites
- **Eclipse IDE:** Used for development and running the project.
- **Java 8 or higher:** Required to compile and run the code. You can download it from Oracle's website.

### Running the Project
1. **Download or Clone the Repository**: Clone or download the project repository to your local machine.
2. **Open the Project in Eclipse**: 
   - Open Eclipse and import the project as an existing Java project.
3. **Run `Main.java`**: 
   - Navigate to the `Main.java` file.
   - Right-click and select "Run As" > "Java Application".
4. **Landing Page**: Once the project is running, you will see a landing page with the name "Titanium Bank" and two buttons: **Login** and **Register**.

### User Flow 🏦

1. **Register**: 
   - Click on the **Register** button.
   - Fill out the registration form with your personal information.
   - After successful registration, you will be shown a success message.
   - Press the **Back** button to return to the landing page.

2. **Login**:
   - Click on the **Login** button.
   - Enter your credentials, including an account number and PIN.
   - Press **Login** to proceed.

3. **Dashboard**: 
   - Once logged in, you will be directed to the main dashboard.
   - The dashboard contains six buttons:
     - **Create Account**: Create a new bank account.
     - **Withdraw**: Withdraw funds from your account.
     - **Transfer Money**: Transfer funds between your accounts.
     - **Deposit**: Deposit funds into your account.
     - **View Accounts**: View a list of your accounts.
     - **View Transaction History**: View your transaction history.

4. **Banking Activities**:
   - After creating an account, you can perform various activities like withdrawing, transferring, depositing, and more.
